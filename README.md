Pizza Sales Analytics Dashboard

An end-to-end business intelligence project built using Power BI and MySQL to analyze pizza sales performance across multiple business dimensions.

Project Overview

This dashboard was designed to transform raw transactional data into meaningful business insights through interactive reporting and data visualization. The project includes data cleaning, KPI development, SQL-based analysis, and dashboard design.

Key Features
Revenue, Orders, and Sales KPI tracking
Monthly sales trend analysis
Order distribution by weekday
Sales breakdown by pizza category and size
Top-performing products analysis
Interactive filters and drill-down reporting
